# dbcgame

## Play on iPhone

You can play `dbcgame` from an iPhone using Safari (or any modern iOS browser) as long as the game is running on a reachable web address.

### If the game is already hosted online
1. Open Safari on your iPhone.
2. Enter the game URL.
3. Tap **Share → Add to Home Screen** to launch it like an app.

### If you are running the game locally on your computer
1. Start the game server on your computer and bind it to all interfaces (`0.0.0.0`) so other devices on your network can connect.
2. Find your computer's local IP address (for example, `192.168.1.20`).
3. Make sure your iPhone is on the same Wi-Fi network.
4. On iPhone Safari, open `http://<your-local-ip>:<port>`.

### iPhone tips
- Keep the phone in landscape mode if the game UI is wide.
- If controls feel zoomed, use Safari's **aA → Website Settings → Request Desktop Website**.
- If it doesn't load, verify firewall/network settings and that the server port is open.
