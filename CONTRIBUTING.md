# Contributing to DBCGame

Thank you for considering contributing to DBCGame! We welcome contributions from everyone.

## How You Can Contribute

### 1. Reporting Bugs or Proposing Features
- Open an issue describing the bug or feature in detail.

### 2. Submitting Code
- Fork the repository and clone your fork.
- Create a new branch for your changes.
- Ensure your code follows the style guide and includes tests if applicable.
- Submit a Pull Request with a clear title and description.

### 3. Improving Documentation
- Help us enhance the project documentation by updating existing files or creating new ones.

## Guidelines

- Be respectful and professional in your interactions.
- Follow the [Code of Conduct](CODE_OF_CONDUCT.md).

Thank you for your contributions!
